# 全栈开发面试指南

> 涵盖前端、Node.js、Go后端、AI Agent开发及经典算法题的全面面试准备资料

## 📖 关于本书

本书是一份全面的技术面试准备指南，旨在帮助开发者系统性地准备技术面试。内容覆盖前端开发、Node.js、Go后端开发、AI Agent开发四大方向，同时包含经典算法题解析。

## 🎯 适用人群

- 准备前端/全栈工程师面试的开发者
- 准备Go后端工程师面试的开发者
- 准备AI/Agent方向面试的开发者
- 希望系统性复习计算机基础知识的工程师

## 📚 内容结构

### 第一部分：前端开发

涵盖HTML/CSS、JavaScript、TypeScript、浏览器原理、网络协议、React、Vue、前端工程化、性能优化、前端安全等核心知识点。

### 第二部分：Node.js开发

涵盖Node.js基础、事件循环、流与文件系统、常用框架（Express/Koa/NestJS）、数据库操作、最佳实践等。

### 第三部分：Go后端开发

涵盖Go语言基础、并发编程（GMP模型）、内存管理与GC、Web框架（Gin/gRPC）、数据库与系统设计、测试与最佳实践等。

### 第四部分：AI Agent开发

涵盖大语言模型基础、Prompt Engineering、RAG检索增强生成、Agent架构设计、工具调用（Function Calling/MCP）、实战与进阶等。

### 第五部分：经典算法题

涵盖数组与字符串、链表、树与图、动态规划、排序与搜索、回溯与贪心等经典算法题，提供JavaScript和Go双语言解法。

## 🚀 如何使用

1. 按照章节顺序系统学习
2. 重点关注面试高频考点（标注为 `>` 的引用块）
3. 动手实践代码示例
4. 结合算法题进行刷题练习
